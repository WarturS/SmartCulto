# CW_Economize

Um projeto criado em partipação do Vibe & Build realizada pela FICR (Faculdade Imaculada Conceição do Recife)

## Objetivo: 
O Projeto tem como finalidade auxiliar o usuário na gestão de seus gastos alimentares mensais 

## Diferencial : 
Diferente das demais aplicações , o nosso sistema permite a criação de um plano mensal personalizado com base no valor que o usuário estipula para gastos no mês. Além de previnir gastos exuberantes , com indicações de gastos no ato da criação do plano mensal, e avisos indicando estar próximo do limite. 

## Ferramentas utilizadas : 
Google Ai Studio, Vercel , Chat GPT
