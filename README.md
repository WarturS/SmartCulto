# SmartCulto

Um projeto criado em partipação do Vibe & Build realizada pela FICR (Faculdade Imaculada Conceição do Recife)

## Objetivo: 
O Projeto tem como finalidade automatizar a tarefa de gerenciar o funcionamento de um Culto, cadastrando visitantes. 

## Requisitos Funcionais :

1 - O sistema deve permitir Login com e-mail e senha; 
2 - O sistema deve permitir redefinição de senha ; 
3 - o sistema deve apresentar como home inicial , algumas funcionalidades : Adicionar um novo culto(Tipo de culto, e Data),  Cultos adicionados (Adicionar Visitante), Perfil (Adm ou Recepcionista)
4 - O sistema deve conter dois tipos de usuário : Adm e Recepcionista; 
5 - O sistema deve permitir a criação de um novo culto; 
6 - O sistema deve permitir na funcionalidade Cultos Adicionados, adicionar os visitantes(Nome , evangélicos ou não evangélicos); 
7 - Ao selecionar se for evangélico, o sistema deve permitir digitar qual congregação faz parte e a denominação; 
8 - O usuário ADM deve somente visualizar os cultos adicionados e os visitantes daquele culto. 
9 - O recepcionista deve visualizar todas as funcionalidades do sistema, assim como todos os visitantes adicionados;

## Requisitos não funcionais: 

1- O sistema deve ser simples e utilizável; 
2- O sistema deve responder a 3,5 segundos entre as transições de uma tela para outra; 
3 - O sistema deve ser intuitivo para com o usuário; 

## Ferramentas utilizadas : 
Google Ai Studio, Chat GPT
## Stacks : 
PHP + JSON
